# BellaNails - Página de salón de uńas

Esta carpeta ahora contiene una página web simple y limpia para un negocio de uńas.

## Archivos

- `index.html`: página principal con secciones de servicios, galería, precios y contacto.
- `style.css`: estilos modernos y responsivos.
- `script.js`: validación básica del formulario y navegación activa.

## Uso

1. Abre `index.html` en el navegador.
2. Edita el contenido si quieres cambiar el nombre, teléfono o servicios.
