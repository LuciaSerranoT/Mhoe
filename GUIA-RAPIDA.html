2️⃣ Configurar Formulario de Contacto

Hay varias opciones. La más sencilla es FormSubmit:

Opción 1: FormSubmit (recomendado - sin configuración)

  1. Ve a FormSubmit.co
  2. En index.html, línea 200, cambia: 
    <form class="contact-form" action="https://formsubmit.co/TU-EMAIL@gmail.com" method="POST">
  3. Reemplaza TU-EMAIL@gmail.com con tu email real
  4. ¡Listo! Recibirás los mensajes en tu email

Opción 2: EmailJS (más profesional)

  1. Regístrate en EmailJS (gratuito)
  2. Copia tu Service ID, Template ID y Public Key
  3. En script.js, añade al inicio: 
    // Inicializar EmailJS
emailjs.init("TU_PUBLIC_KEY");

// Reemplaza contactForm listener
document.querySelector('.contact-form').addEventListener('submit', function(e) {
    e.preventDefault();
    
    emailjs.sendForm('TU_SERVICE_ID', 'TU_TEMPLATE_ID', this)
        .then(() => {
            alert('¡Mensaje enviado correctamente!');
            this.reset();
        })
        .catch(err => alert('Error: ' + err));
});

5️⃣ Publicar tu Portfolio

Opción A: GitHub Pages (recomendado)

  1. Crea una cuenta en GitHub
  2. Crea un nuevo repositorio llamado portfolio (puede ser privado)
  3. Sube tus archivos (index.html, style.css, script.js, README.md)
  4. Ve a Settings → Pages
  5. En "Source", selecciona main branch
  6. Tu sitio estará en: https://tu-usuario.github.io/portfolio
✅ Gratis, sin anuncios, dominio personalizado con custom domain

Opción B: Netlify

  1. Ve a Netlify.com
  2. Haz drag & drop de tu carpeta de proyecto
  3. ¡Automáticamente publicado!
  4. Tu URL será algo como: proyecto-xxxxx.netlify.app
✅ Más fácil aún, SSL automático

Opción C: Dominio personalizado

  1. Compra un dominio en Namecheap o similar
  2. En GitHub Pages o Netlify, añade tu dominio en Settings
  3. Apunta los DNS del dominio al servicio

7️⃣ Elementos Opcionales

Añadir favicon (icono del sitio)

  1. Crea o descarga un icono (logo) en formato .ico o .png
  2. Colócalo en la misma carpeta que index.html
  3. En <head> de index.html, añade: 
    <link rel="icon" type="image/x-icon" href="favicon.ico">

Cambiar foto de perfil

Descarga una imagen profesional y colócala en tu carpeta. Luego puedes añadir una sección de foto.
